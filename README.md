# Kupa Manager Web

ממשק נייד (RTL) לניהול הכנסות, הוצאות, לקוחות ומוצרים — לפריסה ב-Vercel.

## הרצה מקומית

```bash
npm install
npm run dev
```

## סנכרון בין מכשירים

במסך **סנכרון**: שמרו את קוד הסביבה, לחצו **שמור לענן**, ובמכשיר אחר **טען מהענן**.

בפרודקשן נדרש `BLOB_READ_WRITE_TOKEN` (Vercel Blob).
